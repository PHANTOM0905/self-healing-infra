✅ NGINX Service is Running

🧩 Project Components:

  • Prometheus: Metrics collection and alerting
  • Alertmanager: Alert routing and management
  • Grafana: Metrics visualization and dashboards
  • Node Exporter: Host system metrics exporter
  • NGINX: Web service being monitored
  • NGINX Exporter: Exports NGINX metrics to Prometheus
  • cAdvisor: Container resource usage exporter
  • Ansible: Used for automated healing via playbooks
  • Webhook Handler: Python service that receives alerts and triggers Ansible

  • 📊 Prometheus Dashboard
